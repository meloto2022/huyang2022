# 获客文案库 Skill

## 技能说明

本技能用于生成**获客型短视频文案**，专门针对智能家居、灯光设计等本地生活服务行业。

**核心特点：**
- 30-60秒口播长度（150-250字）
- 爆款文案结构（钩子+痛点+解决方案+CTA）
- 贵阳重度本地化（具体楼盘、商圈、案例）
- 多种模板类型（痛点型、对比型、场景型等）

## 触发关键词

当用户使用以下表达时触发本技能：
- "帮我生成一个智能家居的获客文案"
- "写一个关于灯光设计的获客视频脚本"
- "贵阳智能家居行业的获客文案"
- "生成获客文案"
- "智能家居文案"

- "爆款文案模板"

## 工作流程

### 1. 识别用户需求

首先确认用户的具体需求：
- **行业类型**：智能家居 / 灯光设计 / 其他本地服务
- **文案主题**：产品推广、案例展示、避坑指南、知识科普等
- **目标客户**：装修业主、设计师、同行等
- **特殊要求**：是否需要强调具体楼盘、价位段、功能等

### 2. 选择合适的模板

根据用户需求选择模板类型：

**智能家居模板** (`templates/smart-home/`)：
- `pain-point.md` - 痛点型：装修遗憾、智能家居太复杂
- `comparison.md` - 对比型：前后对比效果
- `budget.md` - 预算型：避坑省钱方案
- `scene.md` - 场景型：生活场景演示

**灯光设计模板** (`templates/lighting/`)：
- `before-after.md` - 前后对比型
- `temperature.md` - 色温搭配型
- `mistake.md` - 避坑型

**通用开头模板** (`templates/generic/`)：
- `question.md` - 提问式开头
- `story.md` - 故事式开头
- `data.md` - 数据式开头

### 3. 融入本地化元素

**贵阳楼盘库** (`local-data/guiyang-locations.md`)：
- 高端楼盘：中铁国际、未来方舟、世纪城、观山湖
- 热门小区：花果园、会展城、金融城、白云新区
- 商圈：喷水池、大十字、小十字、观山湖CBD

**本地案例库** (`local-data/guiyang-cases.md`)：
- 贵阳客户真实案例
- 本地工地实拍
- 贵阳气候特殊考虑（阴雨天多）

### 4. 生成符合爆款结构的文案

**文案结构要求**：

```
标题（吸引点击）
├─ 类型1：数字+利益点 → "3个让贵阳业主后悔的智能家居决定"
├─ 类型2：对比反差 → "做智能家居一共5步，但90%业主倒在第2步"
├─ 类型3：本质揭示 → "智能灯光的本质是生活方式"
└─ 类型4：时间效率 → "一周搞定全屋智能，我用了这3招"

开头钩子（前3秒）
├─ 提问式："有没有一种可能..."
├─ 痛点切入："你们肯定想象不到..."
├─ 数据震撼："99%的人都..."
└─ 故事开场："去年有个贵阳业主..."

正文内容（约40秒）
├─ 1. 痛点共鸣 - 描述用户遇到的问题
├─ 2. 案例故事 - 真实案例增加可信度
├─ 3. 方法/观点 - 提供解决方案
├─ 4. 对比验证 - 前后对比，数据支撑
└─ 5. 金句总结 - 提炼核心观点

行动引导（最后10秒）
├─ 引导评论："评论区说说你的情况"
├─ 引导私信："需要方案扣1"
├─ 引导关注："关注我，下期讲..."
└─ 引导到店："贵阳展厅在..."
```

**文案长度规范**：
- 总字数：150-250字
- 口播时长：30-60秒
- 节奏：快、简洁、有力
- 结构：3秒钩子 + 40秒内容 + 10秒CTA + 7秒缓冲

## 心理触发点

生成文案时融入以下心理触发点：

| 触发点 | 应用场景 | 示例 |
|--------|----------|------|
| **恐惧** | 失去机会、做错选择 | "装错后悔一辈子" |
| **认同** | 被理解、被看见 | "你是不是也有这种困扰" |
| **优越感** | 学到别人不知道的 | "90%业主都不知道的秘密" |
| **紧迫感** | 立即行动的需求 | "装修前一定要知道" |
| **稀缺性** | 限定、专属 | "贵阳仅10个名额" |

## 模板使用示例

### 痛点型模板（智能家居）

```markdown
标题：装修完才明白的3个智能家居遗憾，贵阳业主必看

开头：你们肯定想象不到我去年遇到的一个花果园业主...

正文：
1. 痛点：装修完想加智能，全部要拆墙重装
2. 案例：世纪城王先生，多花了3万
3. 方法：装修前先规划这3个点位
4. 对比：有规划和没规划的差价

CTA：贵阳业主装修前免费方案，评论区扣"方案"
```

### 对比型模板（灯光设计）

```markdown
标题：同样花5万做灯光，为什么他家像样板间你家像出租屋

开头：昨天去观山湖一个工地验收，对比太强烈了...

正文：
1. 对比：两个业主同样的预算，效果天差地别
2. 原因：色温、层次、显色指数
3. 方法：3个灯光设计黄金法则
4. 验证：中铁国际实景案例

CTA：需要灯光设计方案的，私信我
```

## 注意事项

1. **真实性**：所有案例必须基于真实情况，避免夸大
2. **本地化**：必须融入贵阳本地元素，不能泛泛而谈
3. **专业性**：体现专业度，但语言要通俗易懂
4. **节奏感**：文案要有节奏感，适合口播
5. **行动点**：每个文案必须有明确的行动引导

## 生成指令

当用户请求生成文案时，按以下步骤操作：

1. **确认需求**：询问行业、主题、目标客户
2. **选择模板**：根据需求选择合适模板
3. **读取数据**：读取本地化数据和案例
4. **生成文案**：按爆款结构生成
5. **输出结果**：输出标题、文案、预计时长、字数统计
